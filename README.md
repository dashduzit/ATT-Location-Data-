# ATT-Location-Data-Analysis - Reproducible Geolocation Verification 

This repository contains certified AT&T geolocation data and interactive visual reports. The dataset is focused on comparing the cliamed location versus verified device location on May 24, 2025.

# Contents 

**Data:** Raw AT&T location data in CSV format.

**Analysis:** Jupyter Notebook used for data analysis and visualization.

**Reports:** HTML and image files providing a clear visual comparison of locations.

# How to Use 

**View the Report:** Simply open the HTML report in any browser to view the interactive map and visualizations. All files are organized for easy reference. 

**Reproducibility:** The Jupyter Notebook (.ipynb) contains all code required to reproduce the analysis and visualizations. Reviewers may download or copy the notebook into their own Jupyter environmnent and run it directly using the included CSV file. 

## How to Reproduce the Analysis

1. Download or open the dataset (att_2025-05-24_converted(in).csv)
2. Open the Jupyter Notebook (.ipynb) in your own Jupyter environment
3. Run all cells to reproduce the cleaning, mapping, and visual outputs
4. Alternatively, open the HTML report directly in a browser to view the interactive map without running any code

## Purpose

This repository was created to provide transparent, reproducible analysis of AT&T device geolocation data for May 24, 2025, including visual maps and distance comparison against claimed location coordinates.
